VSIMM_DDHTA serves as the main function for evaluating the performance of DCJA, DSTNet, and the combined DCJA+DSTNet model.

To run a specific test, please modify lines 26-28 of the script. Set the corresponding line to 1 to activate a test, while any other value will deactivate it.

The included test_3 data corresponds to Test Trajectory 3 as described in our paper, and DDHTA_net contains the trained weights for the model.
